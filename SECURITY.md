# Security Policy

## Supported Versions

We currently support the latest version of SchemaLens. Please ensure you are running the most recent release to receive the latest security updates.

| Version | Supported          |
| ------- | ------------------ |
| v0.1.x  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of SchemaLens seriously. If you believe you have found a security vulnerability, please do **not** open a public issue. Instead, follow these steps:

1. Send a detailed report to the maintainer via GitHub's Private Vulnerability Reporting (if enabled) or contact the project owner directly.
2. Include as much information as possible:
   - Type of issue (e.g., XSS, RCE, data leak).
   - Location of the vulnerability.
   - Steps to reproduce.
   - Potential impact.

We will work with you to resolve the issue.
