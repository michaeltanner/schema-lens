# Code Review Expert Skill

**Goal**  
Deliver thorough, actionable code reviews aligned with senior engineering standards.

**Review Checklist** (always cover all)  
1. Correctness & Edge Cases  
2. Security Vulnerabilities (input validation, auth, injections, secrets)  
3. Performance & Scalability  
4. Separation of Concerns & Architecture  
5. Maintainability (monolithic files, duplication, complexity)  
6. Style, Naming, Comments, Testing  
7. Best Practices for the stack/language  

**Process**  
- Start with positives.  
- List issues by severity (Critical / High / Medium / Low).  
- Provide specific fixes or refactors.  
- Suggest tests where missing.  

**Feedback Style**  
Be constructive, explain "why", and offer alternatives.
